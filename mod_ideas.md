# Hard Mode

Ramp up difficulty of the game. Still need to brainstorm how to do that.

# New Game +

Take 2 monsties from 1 save file and bring them into a new one. This would reset their genes, level and so on. This should also increase difficulty or add more content to the game; this would be a great place to add custom genes.

# Custom Genes

I am a little dissapointed in the depth of the gene pool and would like to add more. Would be cool if this could extend to let users create their own genes.

# Remove Ratha

Already exists on Nexus but isn't optimal. I would like to make a more complete version.

# Level Cap

Prevent the player and monstie's from gaining levels for an added layer of difficulty. Planning two features:

- Set level manually
- Automatically adjust level based on enemy

# Gene Bank

Store genes in an external structure so that you don't need a ton of monsties named "Fodder 1", "Fodder 2", etc.

# Alternate Gene Unlock Condtions

Meant to spice up the game instead of having every gene locked behind a level. These could be triggered by things like story flags, mission flags, a certain number of skill uses, collecting some number of a certain monster, and so on.

# Post-Combat Statistics

Meant to show interesting statistics such as:

- highest damage attack
- hp gained
- number of times you switched monsties
- total turns taken
- kinship attacks used

Could include in-combat notices of how much damage you did, how much you would have done if not capped at 9999, and so on.

# RNG Section

**Stat Fluctuation**

Currently, all base stats are fixed from 1-10 with no total exceeding 31. This could bump up the cap, change weights based on rank, and so on.

**Change Native Monsters**

Randomize what monsters appear where. This could add an element of exitement after discovering each area, as you'll never know which area has which monsters. Might make this truly random but could also limit the RNG, not sure on how I want to do that one yet.

**Weapon Buffs**

Give a chance to increase a weapon's stats after some condition is met. This would eventually make some weapons more competitive and allow for more of them to be used. Would probably cap the stats at the deviljho weapon's stat lines.
